  Isaac Racing Mods

A collection of racing mods for The Binding of Isaac: Afterbirth

Download v3.4.1

What Is It?

This is a program that contains 3 racing mods:

  • The Jud6s Mod
  • Instant Start Mod
  • Diversity Mod

It will also automatically keep everything up to date so that you don't have to be bothered downloading the latest versions over and over.

To use it, simply run the isaac-racing-mods program. You do not have to move any files or uninstall other mods.

What Exactly Does the Program Do?

  • When the program is started, any important files in your "resources" directory are backed up to "resources_tmp##########" (a directory name corresponding to the current time).
  • Once you select the mod you want to play, it moves files into your "resources" directory and automatically starts Isaac.
  • When closed, the program will put everything in your "resources" directory back to the way it was.

Known Issues

  • If the program closes unexpectedly, it can leave mod files installed. To fix this and go back to a "vanilla" game, open the program, select "Miscellaneous Stuff", and click on "Uninstall all existing mods". Or, if you want to do it yourself manually, simply delete all of the files and folders in the Isaac "resources" directory except for the "packed" directory. By default, the "resources" directory is located at:
    C:\Program Files (x86)\Steam\steamapps\common\The Binding of Isaac Rebirth\resources
  • There are no release versions for OS X or Linux. However, the program is written in Python 3.4 and is cross-platform. If you have the technical know how, you can just download the source code and run the Python script directly (see the GitHub repository page).
